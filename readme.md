
# How to start

* Step 1: create juekyll env
https://jekyllrb.com/docs/step-by-step/01-setup/

* Step 2: install
```
bundle
```

* Step 3: local serve
```
bundle exec rake serve
```

# Tasks

* serve: local dev serve
```
bundle exec rake serve
```

* build: build site
```
bundle exec rake build
```
